Indicadores para el estudio de tiempos de producción

Calcular el tiempo promedio de producción es una tarea de gran importancia para una industria productiva ya que a partir este indicador se facilita la toma de decisiones, para ello vamos a calcular el tamaño de muestra ideal para identificar el tiempo promedio con un 95% de confianza y un más o menos 5% de nivel de precisión.
